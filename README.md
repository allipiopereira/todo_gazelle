# Todo API Gazelle

This project is a Todo application with CRUD operations, using the Gazelle framework.

## How to use

```sh
git clone https://github.com/allipiopereira/todo_gazelle
```

```sh
cd todo_gazelle
```

```sh
dart pub get
```

```sh
gazelle run
```

## Gazelle Framework

Github: <https://github.com/intales/gazelle>
